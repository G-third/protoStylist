About proto/Stylist

  • proto/Stylist is a tool for rapidly establishing foundational CSS for web prototypes. It generates core design tokens and barebones styles to accelerate early-stage design.
  • The output is a starting point, not a production-ready design system. It provides a solid baseline that can be extended and refined during detailed design and development.
  • For best results, work "atomically" from the top down. Start by defining your Baseline Styles (Typography, Color, etc.), which serve as the foundation for the Controls & Input and Components further down the list.
  • As you make selections, styles are generated in the Tokens & CSS viewer below. You can export a CSS file at any time or preview your styles in sample layouts using the Style Sampler app.
